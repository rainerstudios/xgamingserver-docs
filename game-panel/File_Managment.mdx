title: "File Management on XGamingServer"
---

Learn how to manage your game server files directly through the control panel or via SFTP.

## Overview

The File Management system allows you to:

- View and edit server files
- Upload/download files
- Create folders and archives
- Set file permissions
- Manage configuration files

## Getting Started

### 

<Steps>
  <Step title="Accessing File Manager">
    1. Log in to [panel.xgamingserver.com](http://panel.xgamingserver.com)
    2. Select your game server from the dashboard
    3. Click "File Manager" in the top bar

       ![File Sv](/SVG/File.svg)
  </Step>
  <Step title="Navigating Files">
    - Click folders to navigate
    - Use breadcrumbs at top to track location
    - Current path: /home/container/
  </Step>
  <Step title="Basic File operations ">
    **To create a new folder**:

    1. Click "New Folder" button
    2. Enter folder name
    3. Click "Create"

    **To upload files:**

    1. Click "Upload" button
    2. Select files from your computer
    3. Wait for completion
  </Step>
  <Step title="Editing files">
    1. Locate the file (e.g., [server.properties](http://server.properties))
    2. Click the file name
    3. Make changes in the editor
    4. Click "Save"
    5. Restart server if needed
  </Step>
</Steps>

## Advanced Features

### 

<AccordionGroup>
  <Accordion title="File Permissions " icon="files">
    1. Right-click any file/folder
    2. Select "Change Permissions"
    3. Set numeric value (e.g., 755 for executables)
    4. Click "Update"

       ![Perm Sv](/SVG/Perm.svg)
  </Accordion>
  <Accordion title="Compressing Files" icon="compress">
    1. Select files/folders with checkboxes
    2. Click "Compress" button
    3. Choose format (ZIP/TAR)
    4. Name the archive
    5. Click "Compress"
  </Accordion>
</AccordionGroup>

## Common File Locations

| Game      | Important Files                             |
| --------- | ------------------------------------------- |
| Minecraft | `server.properties`, `/plugins/`, `/world/` |
| Rust      | `server.cfg`, `/oxide/`, `/save/`           |
| ARK       | `Game.ini`, `/ShooterGame/Saved/`           |
| FiveM     | `server.cfg`, `/resources/`                 |

## Troubleshooting

### File Upload Issues

- Check available disk space
- Verify file permissions
- Ensure correct file format
- Ensure you're not uploading more than 100MB

### Permission Errors

1. Check current permissions
2. Contact support if persists

## Pro Tips

- Always back up before editing
- Use SFTP for bulk transfers
- Compress large files before upload
- Restart server after config changes

For advanced file management, consider using [SFTP](/game-panel/sftp) for faster transfers.
